# slurmviz

A terminal UI dashboard for monitoring Slurm GPU cluster status, built with [Textual](https://github.com/Textualize/textual).

## Features

- **Node map** вҖ” per-node GPU utilization bars with state indicators
- **GPU summary** вҖ” cluster-wide totals and per-user GPU/job breakdown
- **My jobs** вҖ” your running and pending jobs at a glance
- **GPU grid** вҖ” per-GPU utilization bars across the entire cluster
- **Auto-refresh** вҖ” configurable polling interval (default: 5s)
- **Demo mode** вҖ” realistic fake data when Slurm isn't available

## Installation

```bash
uv sync
```

## Usage

```bash
# Live mode (connects to Slurm)
uv run slurmviz

# Demo mode (fake data, no Slurm required)
uv run slurmviz --demo

# Custom refresh interval (seconds)
uv run slurmviz -r 10
```

### Keyboard Shortcuts

| Key | Action              |
|-----|---------------------|
| `q` | Quit                |
| `r` | Refresh immediately |
| `d` | Toggle demo mode    |

## Requirements

- Python 3.10+
- [uv](https://docs.astral.sh/uv/)
- Slurm commands (`sinfo`, `squeue`) for live data вҖ” falls back to demo mode automatically if unavailable
